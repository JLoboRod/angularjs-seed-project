# AngularJS Seed Project

### List of features:

- AngularJS 1.6
- Stateful/stateless and routed components
- One-way dataflow
- Lifecycle hooks
- ui-router
- Karma/Jasmine test suite
- SCSS
- Gulp
- ES6

### Setup and tasks

Just `yarn install`, `yarn start` and ready to go.
Run tests with `npm test`.
